Libraries used : 
QR-Code Generator -> https://www.nayuki.io/page/qr-code-generator-library
Tiny -PNG-Out -> https://www.nayuki.io/page/tiny-png-output

To run it after cloning:

mkdir build
cd build

cmake ..
make all
src/qr-to-png

this will save the qr code of the data as png image
